Preguntas y Respuestas

¿Cada cuanto tiempo se actualizan los datos?

Los datos que se muestran están actualizados, cada vez que se muestra un gráfico o informe se conecta a la base de datos para obtener la información que se muestra.

¿Qué hacer si no se muestran los gráficos y los informes?

La conexión con la base de datos no es posible o el servidor de la base de datos de la compañía no está disponible.

Si el error persiste contacte con soporte.
